I find it easier to remember concepts when I define them in terms of Objects and the Morphisms that act on the objects. Usually a lot of concepts can be reduced to a set \(X\text{,}\) a function \(f\) that acts on \(X\text{,}\) and its image \(Y\text{.}\) To save some brain space, I will define an algorithm to be a pair of functions \((f,g)\text{.}\) Here, \(g:\mathbb{N}\to\mathbb{R}^+\) computes the time it takes \(f\) to compute the output. Models are usually the output of the function \(f\text{.}\)
